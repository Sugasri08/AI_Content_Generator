# 🤖 AI Content Generator

A multi-agent AI system that generates high-quality articles using CrewAI and Groq LLM, with a Streamlit web interface.

---

## ✨ Features

- 🧠 Multi-agent workflow (Research → Write → Edit)
- ⚡ Powered by Groq LLM
- 🌐 Streamlit UI
- 📄 Download as PDF
- 📝 Download as Word file

---

## 🛠️ Tech Stack

Python • CrewAI • Streamlit • Groq • ReportLab • python-docx

---

## 🚀 How it works

User enters a topic → AI agents research, write, and refine → Final article is displayed and downloadable.

---

## ▶️ Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
